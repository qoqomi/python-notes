# 📌 파이썬 기본 문법 & 함수 정리

## 1. def와 일급 객체

- 파이썬의 함수는 **일급 객체(first-class object)** 입니다.
  - 변수에 할당할 수 있고,
  - 다른 함수의 인자로 전달할 수 있으며,
  - 함수의 반환값으로도 사용 가능합니다.

---

## 2. 람다 함수 (lambda)

- **한 줄로 간단하게 표현할 수 있는 함수**
- 보통 짧은 함수를 간단히 작성할 때 사용
- `map()`, `filter()`, `sort()` 등 고차 함수와 함께 자주 활용됨

### 📌 일반 함수 vs 람다 함수

```python
# 일반 함수
def add(x, y):
    return x + y

# 람다 함수
add = lambda x, y: x + y
✅ 람다 함수 사용 이유
코드의 간결성

간단한 변환/조건 처리를 쉽게 적용 가능

3. 제너레이터와 yield
제너레이터(generator): 데이터를 한 번에 다 만들지 않고, 필요할 때마다 하나씩 생성하는 객체

yield: 값을 반환하면서 함수 실행을 잠시 멈추고, 다음 호출 시 이어서 실행됨

📌 for문 vs yield
for문: 이미 만들어진 데이터(리스트, 튜플 등)를 반복해서 꺼냄

yield: 데이터를 필요할 때마다 하나씩 만들어서 반환

👉 비유:

for = 준비된 밥을 한 번에 가져다줌

yield = 주문할 때마다 즉석에서 요리해줌

4. 주요 내장 함수 정리
📍 출력 & 타입 확인
| 함수        | 설명            | 예시 코드                  | 출력 결과             |
| --------- | ------------- | ---------------------- | ----------------- |
| `print()` | 값을 출력         | `print("안녕하세요!")`      | 안녕하세요!            |
| `type()`  | 데이터 타입 확인     | `print(type(10))`      | `<class 'int'>`   |
|           |               | `print(type(3.14))`    | `<class 'float'>` |
|           |               | `print(type("문자"))`    | `<class 'str'>`   |
| `id()`    | 변수의 메모리 주소 확인 | `a = 10; print(id(a))` | (실행 시마다 달라짐)      |


📍 수학 관련 함수
| 함수        | 설명            | 예시 코드                      | 출력 결과 |
| --------- | ------------- | -------------------------- | ----- |
| `abs()`   | 절댓값 반환        | `print(abs(-5))`           | 5     |
|           |               | `print(abs(10))`           | 10    |
| `pow()`   | 거듭제곱 (a^b 계산) | `print(pow(2, 3))`         | 8     |
|           |               | `print(pow(5, 2))`         | 25    |
| `round()` | 반올림           | `print(round(3.14159, 2))` | 3.14  |
|           |               | `print(round(7.6))`        | 8     |


| 함수        | 역할          | 예시                     |
| --------- | ----------- | ---------------------- |
| `ord()`   | 문자 → 유니코드 값 | `ord("A") → 65`        |
| `chr()`   | 유니코드 값 → 문자 | `chr(65) → "A"`        |
| `int()`   | 정수형 변환      | `int("10") → 10`       |
| `float()` | 실수형 변환      | `float("3.14") → 3.14` |
| `str()`   | 문자열 변환      | `str(100) → "100"`     |
```
